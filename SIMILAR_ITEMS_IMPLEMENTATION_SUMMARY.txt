================================================================================
OVERVIEW
================================================================================
Implemented an intelligent product recommendation system that displays similar items on the product detail page. The system uses a scoring algorithm to recommend products based on:
- Category
- Subcategory
- Tags
- Price range
- Popularity/ratings

Benefits:
✓ Improved product discoverability
✓ Enhanced cross-selling
✓ Increased user engagement
✓ Higher average order value
✓ Shows only genuinely similar items (no unrelated products)

================================================================================
KEY CHANGES
================================================================================

---[ 1. BACKEND - Product Model Enhancement ]---
File: backend/model/productModel.js
ADDED FIELDS:
- tags: [String] (e.g., ["floral", "summer"])
- rating: Number (0-5)
- reviewCount: Number
- popularity: Number

---[ 2. FRONTEND - Recommendation Algorithm ]---
File: frontend/src/components/RelatedProduct.jsx
- New scoring system:
  * Category match: +40
  * Subcategory match: +30
  * Tag similarity: +20
  * Price similarity: +15
  * Popularity/Rating: +10
- FILTER CRITERIA (FIXED):
  * Minimum score: 20 (prevents unrelated products)
  * Requires: Category match OR tag overlap
  * This makes "No similar products found" state reachable
- Returns top 4 genuinely similar products

---[ 3. FRONTEND - Product Detail Update ]---
File: frontend/src/pages/ProductDetail.jsx
- Passes tags and price to RelatedProduct for better recommendations

---[ 4. DATABASE - Dummy Data Script ]---
File: backend/seedDummyData.js
- Seeds 12 products with tags, ratings, popularity for testing

================================================================================
HOW IT WORKS
================================================================================
- User views a product (e.g., "Floral Summer Dress")
- System scores all other products based on similarity
- Products are filtered to ensure minimum score of 20
- Products must have either category match OR tag overlap
- Top 4 genuinely similar products shown in "Similar Items" section
- If no similar products meet the criteria, shows "No similar products found"
- Recommendations update dynamically per product
